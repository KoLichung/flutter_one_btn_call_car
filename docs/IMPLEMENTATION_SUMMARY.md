## 設定需求

### 必要步驟
1. **安裝 Flutter SDK** (3.9.2+)
2. **取得 Google Maps API Key**
3. **配置 Android**:
   - 編輯 `android/app/src/main/AndroidManifest.xml`
   - 替換 API Key
4. **配置 iOS**:
   - 編輯 `ios/Runner/AppDelegate.swift`
   - 替換 API Key
5. **安裝依賴**: `flutter pub get`
6. **執行應用程式**: `flutter run`

### 詳細設定指南
請參考 `SETUP.md` 檔案獲取完整的設定步驟。
